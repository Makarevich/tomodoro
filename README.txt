== Tomodoro ==

Tomodoro is a (yet another) simple timer for usage with the pomodoro technique (see: http://en.wikipedia.org/wiki/Pomodoro_Technique ).

It is inteded to be pretty configurable.

<b>The work is still in progress.</b>
