== Tomodoro ==

Tomodoro is a (yet another) simple timer for usage with <a href='http://en.wikipedia.org/wiki/Pomodoro_Technique'>pomodoro technique</a>.

It is inteded to be pretty configurable.

<b>The work is still in progress.</b>
